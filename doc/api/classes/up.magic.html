up.awaken

Defined in lib/assets/javascripts/up/magic.js.coffee:38

Syntax

up.awaken

(
  • selector
  • awakener
)

Summary

Registers a function to be called whenever an element with the given selector is inserted into the DOM through Up.js.

Parameters:

  • selector String

    The selector to match.

  • awakener Function

    The function to call when a matching element is inserted. The function takes the new element as the first argument (as a jQuery object). It may return another function that destroys the awakened object when it is removed from the DOM, by clearing global state such as time-outs and event handlers bound to the document.

Syntax

up.on

(
  • events
  • selector
  • behavior
)

Summary

Binds an event handler to the document, which will be executed whenever the given event is triggered on the given selector.

Parameters:

  • events String

    A space-separated list of event names to bind.

  • selector String

    The selector an on which the event must be triggered.

  • behavior Function

    The handler that should be called.
