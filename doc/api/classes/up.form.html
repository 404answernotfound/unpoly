up.observe

Defined in lib/assets/javascripts/up/form.js.coffee:76

Syntax

up.observe

(
  • fieldOrSelector
  • options.change
  • [options.frequency=500]
)

Summary

Observes an input field by periodic polling its value. Executes code when the value changes.

This is useful for observing text fields while the user is typing, since browsers will only fire a change event once the user blurs the text field.

Parameters:

  • fieldOrSelector Element | JQuery | String
  • options.change Function | String

    The callback to execute when the field's value changes. If given as a function, it must take two arguments (value, $field). If given as a string, it will be evaled as Javascript code in a context where (value, $field) are set.

  • [options.frequency=500] Number optional

Example:

up.observe('input', { change: function(value, $input) { 
                                              up.submit($input) 
                                            } });

up.submit

Defined in lib/assets/javascripts/up/form.js.coffee:9

Syntax

up.submit

(
  • formOrSelector
  • [options.target]
  • [options.failTarget]
  • [options.history=true]
  • [options.transition]
  • [options.failTransition]
)
Promise

Summary

Submits a form using the Up.js flow.

Parameters:

  • formOrSelector Element | JQuery | String

    A reference or selector for the form to submit. If the argument points to an element that is not a form, Up.js will search its ancestors for the closest form.

  • [options.target] String optional
  • [options.failTarget] String optional
  • [options.history=true] Boolean | String optional

    Successful form submissions will add a history entry and change the browser's location bar if the form either uses the GET method or the response redirected to another page (this requires the upjs-rails gem). If want to prevent history changes in any case, set this to false. If you pass a String, it is used as the URL for the browser history.

  • [options.transition] String optional
  • [options.failTransition] String optional

Returns:

Promise:

A promise for the AJAX response

Example:

up.submit('form')
