up.follow

Defined in lib/assets/javascripts/up/link.js.coffee:29

Syntax

up.follow

(
  • link
  • [options.target]
  • [options.transition]
)

Summary

Follows the given link and replaces a selector in the current page with corresponding elements from a new page fetched from the server.

Parameters:

  • link Element | JQuery | String

    An element or selector which resolves to an <a> tag or any element that is marked up with an up-follow attribute.

  • [options.target] String optional

    The selector to replace. Defaults to the up-target attribute on link, or to body if such an attribute does not exist.

  • [options.transition] Function | String optional

    A transition function or name.

up.visit

Defined in lib/assets/javascripts/up/link.js.coffee:11

Syntax

up.visit

(
  • url
  • options
)

Summary

Visits the given URL without a full page load. This is done by fetching url through an AJAX request and replacing the current <body> tag with the response's <body> tag..

Parameters:

  • url String

    The URL to visit.

  • options Object

    See options for up.replace.

Example:

up.visit('/users')
